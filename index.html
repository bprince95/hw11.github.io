Summary: Latitude vs. X

The purpose of this assignment was to analyze how weather changes as you approach the equator. We first used CityPy to randomly pick 500 cities. We took these 500 cities and crossreferenced them with the the data that is available on the OpenWeather Map API. We wrote code to pull the data from the API and cleaned the data using pandas. After that, we created visualizations that mapped the data using MatplotLib.
